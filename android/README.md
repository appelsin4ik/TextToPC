# 📱 TextBridge Android

> Send text from your phone to your PC instantly.

## ✨ Features

- ✍️ Simple text input
- ⚡ Instant send over Wi-Fi
- 🎨 Clean modern UI
- 🔗 Works with desktop TextBridge app
- 📡 No accounts, no login

## 🚀 Setup

### 1. Open Project

Open the `android/` folder in **Android Studio**


### 2. Build APK

```text
Build → Build APK(s)
```

APK will be located at: ( __app/build/outputs/apk/debug/__ )


### 3. Install

- Transfer APK to your phone
- Install it (allow unknown sources)


## ⚙️ Configuration

Set your PC IP address in the app:

Example:
```text
192.140.0.111:5000
```

**Make sure:**

- PC is running the desktop app
- both devices are on the same Wi-F

## 🧪 Troubleshooting

### ❌ Failed to connect

Check:
- PC app is running
- correct IP address
- same network
- firewall not blocking port 5000
